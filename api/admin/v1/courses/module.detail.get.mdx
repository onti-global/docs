# GET Module Detail

Returns a single module with its variants, each hydrated with its content items. This is the endpoint the module editor loads on open.

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` or `"analyst"` in the JWT.
- Uses the Supabase **admin** client (service-role key); requests bypass RLS entirely.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Accept` | `application/json` | Yes |

## Path parameters

| Name | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | The module id. |

## Response body

`200 OK`

  "message": "",
  "data": {

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Module id. |
| `course_id` | `uuid` | Parent course id. |
| `title` | `string` | Module title. |
| `description` | `string \| null` | Module description. |
| `position` | `integer` | Sort order within the course. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |
| `variants` | `Variant[]` | Module variants, each with its `contents` array. |

### Variant object

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Variant id. |
| `module_id` | `uuid` | Parent module id. |
| `name` | `string` | Variant name. |
| `description` | `string \| null` | Variant description. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |
| `contents` | `Content[]` | Content items belonging to the variant. |

### Content object

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Content item id. |
| `variant_id` | `uuid` | Parent variant id. |
| `type` | `"html" \| "video" \| "audio" \| "meditation" \| "quiz"` | Content type. |
| `title` | `string` | Content item title. |
| `url` | `string \| null` | Media asset URL. |
| `body` | `string \| null` | HTML markup or quiz JSON schema. |
| `position` | `integer` | Sort order within the variant. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry an allowed `admin_dashboard_role` (`admin` or `analyst`). |
| `404` | `module_not_found` | Module does not exist. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "status": 404,
  "message": "Module not found",
  "error": "module_not_found",
  "data": null
}
```
