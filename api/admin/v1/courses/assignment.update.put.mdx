# PUT Override Variant Assignment

<Warning>Planned — this endpoint is not implemented in the backend yet.</Warning>

Manually overrides a learner's variant assignment for a module. Use this to correct an incorrect automatic assignment or to grant a learner access to a specific variant regardless of their assessment answers.

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` or `"analyst"` in the JWT.
- Uses the Supabase **admin** client (service-role key); requests bypass RLS entirely.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Content-Type` | `application/json` | Yes |
| `Accept` | `application/json` | Yes |

## Path parameters

| Name | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | The assignment id (`course_alumni_assignments.id`). |

## Request body

```json
{
  "variant_id": "v3...ii"
}
```

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `variant_id` | `uuid` | Yes | The variant to assign. Must belong to the same module as the existing assignment. |

## Response body

`200 OK`

```json
{
  "status": 200,
  "message": "",
  "data": {
    "id": "a1...mm",
    "user_id": "9c0a...77",
    "module_id": "m1...cc",
    "variant_id": "v3...ii",
    "assigned_at": "2026-06-29T14:00:00Z",
    "created_at": "2026-06-29T10:05:00Z",
    "updated_at": "2026-06-29T14:00:00Z"
  }
}
```

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry an allowed `admin_dashboard_role` (`admin` or `analyst`). |
| `400` | `invalid_body` | `variant_id` is missing or the target variant does not belong to the assignment's module. |
| `404` | `not_found` | Assignment does not exist. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "status": 400,
  "message": "Validation failed",
  "error": "variant_id does not belong to the module of this assignment",
  "data": null
}
```
