# GET List Courses

Returns a paginated list of courses, optionally filtered by lifecycle status. Pagination is cursor-based: pass the `cursor` from a previous response to fetch the next page.

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` or `"analyst"` in the JWT.
- Uses the Supabase **admin** client (service-role key); requests bypass RLS entirely.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Accept` | `application/json` | Yes |

## Query parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `status` | `"draft" \| "published" \| "archived"` | No | Filter by lifecycle status. Omit to return all. |
| `limit` | `integer` | No | Max results per page. `1`–`100`. Defaults to `20`. |
| `cursor` | `uuid` | No | Pagination cursor from a previous response. |

## Response body

`200 OK`

```json
{
  "status": 200,
  "message": "",
  "data": {
    "items": [
      {
        "id": "3cc4...e9",
        "title": "Advanced Facilitation",
        "description": "A deep dive into group facilitation techniques.",
        "status": "published",
        "created_at": "2026-06-29T12:00:00Z",
        "updated_at": "2026-06-29T13:00:00Z"
      }
    ],
    "cursor": null
  }
}
```

| Field | Type | Description |
| --- | --- | --- |
| `items` | `Course[]` | Courses matching the filter. |
| `cursor` | `string \| null` | Cursor for the next page. `null` when there are no more results. |

### Course object

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Course id. |
| `title` | `string` | Course title. |
| `description` | `string \| null` | Course description. |
| `status` | `"draft" \| "published" \| "archived"` | Lifecycle state. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry an allowed `admin_dashboard_role` (`admin` or `analyst`). |
| `400` | `validation_error` | `status` is not a valid enum value, `limit` is outside `1`–`100`, or `cursor` is not a valid UUID. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "status": 400,
  "message": "limit must be an integer between 1 and 100",
  "error": "validation_error",
  "data": null
}
```
