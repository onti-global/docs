# PUT Update Module

Updates a module's title, description, or position. All fields are optional — only provided fields are updated. To reorder several modules at once, prefer the [reorder endpoint](/api/admin/v1/courses/module.reorder.put).

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` or `"analyst"` in the JWT.
- Uses the Supabase **admin** client (service-role key); requests bypass RLS entirely.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Content-Type` | `application/json` | Yes |
| `Accept` | `application/json` | Yes |

## Path parameters

| Name | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | The module id. |

## Request body

```json
{
  "title": "Module 3 — Emotional Regulation (Revised)",
  "description": "Updated description.",
  "position": 1
}
```

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | No | New module title. Must be non-empty if provided. |
| `description` | `string \| null` | No | New description. Pass `null` to clear it. |
| `position` | `integer` | No | New sort order within the course. |

## Response body

`200 OK` — returns the updated module.

```json
{
  "status": 200,
  "message": "Module updated",
  "data": {
    "id": "m3...hh",
    "course_id": "3cc4...e9",
    "title": "Module 3 — Emotional Regulation (Revised)",
    "description": "Updated description.",
    "position": 1,
    "created_at": "2026-06-29T12:10:00Z",
    "updated_at": "2026-06-29T13:00:00Z"
  }
}
```

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Module id. |
| `course_id` | `uuid` | Parent course id. |
| `title` | `string` | Module title. |
| `description` | `string \| null` | Module description. |
| `position` | `integer` | Sort order within the course. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry an allowed `admin_dashboard_role` (`admin` or `analyst`). |
| `400` | `validation_error` | A provided field value is invalid (e.g. empty title, negative position). |
| `404` | `module_not_found` | Module does not exist. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "status": 400,
  "message": "title must be a non-empty string",
  "error": "validation_error",
  "data": null
}
```
