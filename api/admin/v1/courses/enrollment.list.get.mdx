# GET Enrollments (Admin)

Returns enrollments, optionally filtered by course and/or status. Includes basic user identity for display in the dashboard. Results are ordered by `enrolled_at` descending and capped at 200 rows.

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` or `"analyst"` in the JWT.
- Uses the Supabase **admin** client (service-role key); requests bypass RLS entirely.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Accept` | `application/json` | Yes |

## Query parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `course_id` | `uuid` | No | Filter by course. Omit to return enrollments across all courses. |
| `status` | `"active" \| "completed" \| "dropped"` | No | Filter by enrollment status. Omit to return all. |

## Response body

`200 OK`

```json
{
  "status": 200,
  "message": "",
  "data": {
    "items": [
      {
        "id": "e9f0...12",
        "user_id": "9c0a...77",
        "course_id": "1aa2...d4",
        "status": "active",
        "enrolled_at": "2026-06-29T10:00:00Z",
        "completed_at": null,
        "created_at": "2026-06-29T10:00:00Z",
        "updated_at": "2026-06-29T10:00:00Z",
        "user": {
          "email": "ana@example.com",
          "display_name": "Ana García"
        }
      }
    ]
  }
}
```

| Field | Type | Description |
| --- | --- | --- |
| `items` | `Enrollment[]` | Enrollments matching the filters. Ordered by `enrolled_at` descending. Max 200 rows. |

### Enrollment object

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Enrollment id. |
| `user_id` | `uuid` | Enrolled user id. |
| `course_id` | `uuid` | Course id. |
| `status` | `"active" \| "completed" \| "dropped"` | Enrollment lifecycle state. |
| `enrolled_at` | `timestamptz` | When the user was enrolled. |
| `completed_at` | `timestamptz \| null` | When completion was recorded. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |
| `user` | `object` | Basic user identity: `{ email: string \| null, display_name: string \| null }`. |

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry an allowed `admin_dashboard_role` (`admin` or `analyst`). |
| `400` | `validation_error` | `status` is not a valid enum value or `course_id` is not a valid UUID. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "status": 400,
  "message": "status must be one of: active, completed, dropped",
  "error": "validation_error",
  "data": null
}
```
