# POST Create Course

Creates a new course in `draft` status. The course is not visible to learners until its status is transitioned to `published` via the update endpoint.

## Authentication

- Requires a valid Supabase Auth session with `admin_dashboard_role = "admin"` in the JWT.
- The role claim is set by the `admin_dashboard_jwt_hook` at token issuance time.

## Headers

| Header | Value | Required |
| --- | --- | --- |
| `Authorization` | `Bearer <token>` | Yes |
| `Content-Type` | `application/json` | Yes |
| `Accept` | `application/json` | Yes |

## Request body

```json
{
  "title": "Advanced Facilitation",
  "description": "A deep dive into group facilitation techniques."
}
```

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | Yes | Course title. Must be non-empty. |
| `description` | `string` | No | Optional marketing description. |

## Response body

`201 Created`

```json
{
  "id": "3cc4...e9",
  "title": "Advanced Facilitation",
  "description": "A deep dive into group facilitation techniques.",
  "status": "draft",
  "created_at": "2026-06-29T12:00:00Z",
  "updated_at": "2026-06-29T12:00:00Z"
}
```

| Field | Type | Description |
| --- | --- | --- |
| `id` | `uuid` | Course id. |
| `title` | `string` | Course title. |
| `description` | `string \| null` | Course description. |
| `status` | `"draft" \| "published" \| "archived"` | Lifecycle state. Always `"draft"` on creation. |
| `created_at` | `timestamptz` | Row creation timestamp. |
| `updated_at` | `timestamptz` | Row last-modified timestamp. |

## Errors

| Status | Code | When |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing or invalid JWT. |
| `403` | `forbidden` | JWT does not carry `admin_dashboard_role = "admin"`. |
| `400` | `invalid_body` | `title` is missing or empty. |
| `500` | `internal_error` | Unexpected database failure. |

```json
{
  "error": {
    "code": "invalid_body",
    "message": "title is required and must be a non-empty string"
  }
}
```
