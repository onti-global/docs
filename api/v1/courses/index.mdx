---
title: "Courses"
description: "Services for course delivery, enrollment, progress, and back-office management."
---

## User-facing

### Course Catalog
Browse and retrieve published courses.
```
GET  /v1/courses
GET  /v1/courses/:id
```

### Enrollment
Enroll in a course and view enrollment status.
```
POST /v1/enrollments
GET  /v1/enrollments
```

### Course Onboarding
Submit the intake questionnaire attached to a course at enrollment time.
```
GET  /v1/courses/:id/onboarding
POST /v1/courses/:id/onboarding
```

### Assessment
Load and submit an assessment flow. Completed responses trigger variant assignment.
```
GET  /v1/assessments/:id
POST /v1/assessments/:id/responses
```

### Course Player
Fetch a course's modules and the learner's assigned variant contents.
```
GET  /v1/courses/:id/modules
GET  /v1/variants/:id/contents
```

### Progress
Mark content as started or completed.
```
POST /v1/progress
GET  /v1/courses/:id/progress
```

### Answers
Submit and retrieve answers for quiz or interactive content.
```
POST /v1/contents/:id/answers
GET  /v1/contents/:id/answers
```

## Back-office

### Course Builder
Create and manage courses, modules, variants, and content items.
```
POST  /internal/v1/admin/courses
PUT   /internal/v1/admin/courses/:id
POST  /internal/v1/admin/courses/:id/modules
POST  /internal/v1/admin/modules/:id/variants
POST  /internal/v1/admin/variants/:id/contents
PUT   /internal/v1/admin/contents/:id
```

### Enrollment Manager
List and manually manage student enrollments.
```
GET   /internal/v1/admin/courses/:id/enrollments
POST  /internal/v1/admin/enrollments
PATCH /internal/v1/admin/enrollments/:id
```

### Assessment Builder
Build assessment flows and their questions.
```
POST  /internal/v1/admin/assessment-flows
POST  /internal/v1/admin/assessment-flows/:id/questions
PUT   /internal/v1/admin/assessment-flows/:id/questions/:qid
```

### Assignment Rules
Configure the policies and rules that auto-assign variants from assessment answers.
```
POST  /internal/v1/admin/variant-policies
POST  /internal/v1/admin/variant-policies/:id/rules
PUT   /internal/v1/admin/variant-policies/:id/rules/:rid
```

### Variant Assignments
View and manually override a student's variant assignment per module.
```
GET   /internal/v1/admin/courses/:id/assignments
PUT   /internal/v1/admin/assignments/:id
```

### Student Insights
Read insight memory records generated during course content interactions.
```
GET   /internal/v1/admin/students/:id/insights
GET   /internal/v1/admin/contents/:id/insights
```
